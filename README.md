# works-show

个人作品展示，包含个人以及参与的公司项目；

本项目主要用于记录个人项目经历，若有侵权，联系删除！

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
